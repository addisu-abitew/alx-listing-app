# ALX Listing App

## Project Description
The ALX Listing App is a property listing project inspired by Airbnb.  
It aims to provide a clean, reusable, and component-based structure for displaying property information, allowing users to browse and interact with property listings efficiently.

## Project Structure

### Components
- `components/common/Card.tsx`:  
  A reusable Card component to display property information such as images, title, price, and other details.
- `components/common/Button.tsx`:  
  A reusable Button component for actions like "Book Now," "Details," etc.

### Interfaces
- `interfaces/index.ts`:  
  Contains TypeScript interfaces used across the project. Includes placeholders for `CardProps` and `ButtonProps`.

### Constants
- `constants/index.ts`:  
  Stores reusable data, strings, configuration settings, or API URLs used across the project.

### Assets
- `public/assets/`:  
  Stores all static assets such as images and SVGs used in the project.  
  Assets are organized for easy access and reusability
